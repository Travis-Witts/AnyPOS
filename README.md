## Overview
A Mobile responsive react web-application designed for businesses of all sizes to use to track stock-levels, profit-margin and handle cash processing.

- TypeScript
- JavaScript
- ESLint AirBnB Configuration
- SASS
- React
- Express
- MySQL
- Sequelize
- Bcrypt
- Axios
- BootStrap
- Figma
